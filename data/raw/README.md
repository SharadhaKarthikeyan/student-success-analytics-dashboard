# Raw Data Directory

Please place the real, public UCI dataset file here as `student_success_uci.csv`.

## Dataset Information

- **Dataset Name:** Predict Students' Dropout and Academic Success
- **Source:** UCI Machine Learning Repository
- **URL:** [UCI Machine Learning Repository - Predict Students' Dropout and Academic Success](https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success)
- **Format:** CSV
- **Required File Name:** `student_success_uci.csv`

## Download Instructions

1. Visit the dataset page link above.
2. Click **Download** (or download the `dataset.zip` file which contains `data.csv`).
3. Extract `data.csv` if it is zipped.
4. Rename `data.csv` (or the extracted CSV file) to `student_success_uci.csv`.
5. Place the renamed file in this directory (`data/raw/`).

## Dataset Purpose
This dataset is used for analyzing academic outcomes (Dropout, Enrolled, Graduate) based on student demographics and performance metrics.
